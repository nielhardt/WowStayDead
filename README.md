# Stay Dead
Wow addon: Stay Dead

## Description
Stay Dead is a World of Warcraft addon that prevents accidental spirit releases after death. It adds a configurable delay and/or key modifier requirement before you can release your spirit, helping avoid mistakes in dungeons, raids, and other content.

## Features
- Prevents accidental spirit release with a timer and/or key modifier
- Works in dungeons, raids, battlegrounds, open world, and more
- Optionally blocks Soulstone/Reincarnation buttons
